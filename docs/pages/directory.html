Directory functions

The gslab_make library provides functions to make modifications to a directory. Functions to remove files and directories, clear directories, and zip/unzip files are included.

remove_dir(dir_list, quiet=False)

Remove directories in list dir_list using system command. Safely removes symbolic links. Directories can be specified with the * shell pattern (see here).

Parameters:

  • dir_list (str, list) – Directory or list of directories to remove.

  • quiet (bool, optional) – Suppress printing of directories removed. Defaults to False.

Return type:

None

Example

The following code removes directories dir1 and dir2.

remove_dir(['dir1', 'dir2'])

The following code removes directories beginning with dir.

remove_dir(['dir1*'])
clear_dir(dir_list)

Clears all directories in list dir_list using system command. Safely clears symbolic links. Directories can be specified with the * shell pattern (see here).

Note

To clear a directory means to remove all contents of a directory. If the directory is nonexistent, the directory is created, unless the directory is specified via shell pattern.

Parameters:

dir_list (str, list) – Directory or list of directories to clear.

Return type:

None

Example

The following code clears directories dir1 and dir2.

clear_dir(['dir1', 'dir2'])

The following code clears directories beginning with dir.

clear_dir(['dir*'])
unzip(zip_path, output_dir)

Unzips file zip_path to directory output_dir.

Parameters:

  • zip_path (str) – Path of file to unzip.

  • output_dir (str) – Directory to write outputs of unzipped file.

Return type:

None

zip_dir(source_dir, zip_dest)

Zips directory source_dir to file zip_dest.

Parameters:

  • source_dir (str) – Path of directory to zip.

  • zip_dest (str) – Destination of zip file.

Return type:

None
